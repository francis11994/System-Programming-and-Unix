CSc 352: Word Count

Write a C program wordCount, to print a frequency count of all the words coming from input from stdin. The list of words and their counts will be listed in alphabetical (lexigraphical) order as specified below:

  • Input:

    Text from stdin. The text will be limited in size to no more than 1024 words (including repeats).

  • Definition of words

    Words are strings that are separated by white space. Therefore, you can (and should) use scanf to read in each string as a word. However, we must do some further processing. Imagine the input includes the line: "Cats rained down on other cats inlucding the purple cats." Notice that the following words appear in the input "Cats", "cats", and "cats." We would like to count them all as the same word, "cats". To do this we will process every string separated by white space as follows:

    1. Convert all the uppercase letters to lowercase.
    2. Strip all the nonalphabetic symbols from the front of the string.
    3. Strip all the nonalphabetic symbols from the end of the string.

    Following this process "cats!" becomes "cats", "**WARNING**" becomes "warning", "Planet9" becomes "planet". This is not a perfect solution. For example "you're" stays "you're", but we don't want the program to be too complex. Notice that this procedure can leave you with an empty string if the string has no alphabetic characters. For example, "99" and "&" both become "". Your program should ignore such strings. They are NOT concedered to be errors in the input, nor should they be included in the word count.

  • Output:

    After all the input has been read, your program should print out the list of words together with the number of times each appeared. Each word and count should be on its own line. Use the following print statement:

    printf("%s %d\n", word, num)
    where word is the word and num is the number of times it appeared in the input.

  • Assumptions:

    You can assume that each of the words read from the input is at most 128 characters long and that the input contains no more than 1024 words including repeats.

    For example the input below:
    I am the eggman. You are the eggman. I am the Walrus.
    contains 12 words and its output should be:
    am 2
    are 1
    eggman 2
    i 2
    the 3
    walrus 1
    you 1

  • Error Conditions:

    none

  • Makefile:

    In addition to your source files, you should submit a make file named Makefile that supports at least the following functionality:

    make wordCount
    Compiles the C source code to create an executable named wordCount. The compiler options used should include -Wall.

    make coverage (phony target)
    Compiles the source program with the gcc options needed by the gcov tool, namely, "-fprofile-arcs -ftest-coverage". to create an executable named wordCount.

    make clean (phony target)
    Deletes all object files (*.o), if any, as well as the executable wordCount and files generated by gcov (*.gcov, *.gcno, *.gcda).

    Note: the 'C' in wordCount is uppercase. All the other letters are lowercase.

  • Submitting your assignment:

    Collect all of the files related to this assignment, as described below, into a directory named wordCount and use the turnin command to submit this directory (this will cause all of the files in the directory to be submitted). The directory wordCount should contain:

    • the C source code for your program;
    • A makefile, as described above; and
    • a directory named tests that contains a set of inputs for your program that meets the following requirements:
      • each distinct input is in a different file; and
      • collectively, they achieve at least 100% code coverage for your program (as measured by gcov).

  • Suggested Approach:

    Since the input has a maximum size in terms of number of words, you might want to use an array to store all the processed words. After all the words are input, you can sort them. This should make it easy to count how many times each word occurs. There are several built in functions that can be of use in this assignment. For example use the man page to look at tolower, isalpha, strcmp, and strdup.

  • Example: Suppose the input consists of the following lines:
    It was the best of times,
    it was the worst of times,
    Then the output should be
    best 1
    it 2
    of 2
    the 2
    times 2
    was 2
    worst 1
