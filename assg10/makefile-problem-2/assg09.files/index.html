The purpose of this assignment is to continue working with dynamic memory allocation and to learn to write some simple makefiles.

General Requirements

  1. Your C code should adhere to the coding standards for this class. In particular, your programs should not give rise to any errors when run under Valgrind. Also, Your program should free all dynamically allocated memory before exiting. This can also be checked with Valgrind.

  2. Normal execution should be indicated via an exit status of 0. If any errors are encountered during execution, this should be indicated via an exit status of 1.

  3. This document specifies what you can assume without checking and what you should check.
